# Demo-Transaction-API-jmeter

## Project Summary:
This project has demo transaction API test using JMETER

## Technology used:
- JMETER

## Pre requisites:
- Apache JMETER

## How to run
- ``` git clone ```
- ``` npm i ```
- ``` npi test ```

A report will be generate in your project source root
